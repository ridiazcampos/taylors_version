Caracterización de dominio

En la visualización presentada, se tienen datos sobre álbumes y canciones de Taylor Swift, a partir de valores como plays de cada canción, y features que nos entregan información adicional. Los datos fueron obtenidos desde Spotify. Por un lado, la cantidad de plays de cada canción se obtuvo a partir de un escaneo de la aplicación, y su lectura a partir de python. Las features, número de canción, entre otras, fueron obtenidas a partir de la API de spotify
